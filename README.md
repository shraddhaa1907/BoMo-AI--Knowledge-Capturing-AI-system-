# 🧠 BoMo-AI: Knowledge Capturing AI System

## 📌 Overview

BoMo-AI is an AI-powered knowledge capturing system designed to extract, structure, and visualize insights from unstructured documents, including multilingual data such as Japanese meeting notes.

The system transforms raw textual data into meaningful Knowledge Graphs, enabling better understanding, retrieval, and decision-making.

---

## 🚀 Features

* 📄 Extracts text from documents (PDFs/images)
* 🌏 Supports multilingual input (Japanese included)
* 🧠 NLP-based entity and relationship extraction
* 🔗 Builds dynamic Knowledge Graphs
* 📊 Graph visualization for insights
* 🔍 Enables semantic search and knowledge retrieval

---

## 🏗️ Tech Stack

* Python
* NLP: spaCy / Hugging Face Transformers
* Japanese Processing: MeCab
* Graph Processing: NetworkX
* Visualization: Matplotlib / PyVis
* Optional: Neo4j (Graph Database)

---

## ⚙️ How It Works

1. Input document (Japanese/English)
2. Text extraction and preprocessing
3. Tokenization and entity recognition
4. Relationship extraction
5. Knowledge Graph construction
6. Visualization of insights

---

## 📷 Example Workflow

Input → Japanese Meeting Notes
Output → Structured Knowledge Graph with entities and relationships
